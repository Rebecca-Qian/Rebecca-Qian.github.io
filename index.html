I created a Ray Tracing program that renders ppm image representations of 3D shapes, such as spheres and cubes. The Ray Tracer projects patterns onto shapes, scales images according to depth and shows intersecting objects.

Cache Simulation

I programmed a Cache Memory Simulator that tracks Cache hits, misses, cold misses and follows the LRU replacement policy. My implementation uses a linked list of two-dimensional arrays to simulate Cache memory blocks and sets. My Cache simulation passed the performance efficiency test with 100%.
